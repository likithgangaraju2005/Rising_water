# 🌊 Flood Prediction System (Machine Learning + Flask)

## 📌 Overview
This project is a Flood Prediction System that uses Machine Learning to predict flood risk based on environmental factors like rainfall, river level, temperature, and humidity. It also includes a Flask web application for user interaction.

---

## 🚀 Features
- Predict flood risk using ML model
- Simple web interface using Flask
- Fast and accurate predictions
- Separate testing script for verification
- Easy and user-friendly UI

---

## 🧠 Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- Pandas
- HTML, CSS, JavaScript
